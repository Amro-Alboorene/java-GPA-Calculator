# Java GPA Calculator

## Description
This is a console-based Java program to calculate student GPA, determine pass/fail status, and assign letter grades (A–E).
Developed as part of Software Engineering studies at Zarqa University.

## Features
- User inputs number of courses and grades
- GPA calculation
- Pass/Fail determination
- Letter grade classification (A, B, C, D, E)
- Showing Max and Min grades
